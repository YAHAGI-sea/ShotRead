# ShotRead (SOKUDO-KUN)

> このプロジェクトは、**AntigravityのAgent Manager Mode** を使用して作成されました。

**ShotRead** (旧名: SOKUDO-KUN) は、テキストを効率的に読むために設計されたインスタント速読Webアプリケーションです。文章を読みやすい単位（チャンク）に分割し、設定した速度で順次表示することで、眼球運動を最小限に抑え、リーディング速度を向上させます。

![ShotRead Screenshot](https://via.placeholder.com/800x400?text=ShotRead+Screenshot+Placeholder)

## 主な機能

-   **粒度調整 (Granularity)**: テキストの分割単位を「単語レベル」から「文レベル」まで、5段階で調整可能です。
-   **スマート再生制御**:
    -   **位置保持**: 読書中に粒度設定を変更しても、現在の読書位置を維持したまま一時停止します。
    -   **自動一時停止**: モード変更時などは自動的に一時停止し、見失うのを防ぎます。
-   **2つの閲覧モード**:
    -   **Auto (Timer)**: 設定した速度（ミリ秒/チャンク）で自動再生します。
    -   **Manual**: キーボードやボタン操作で、自分のペースで読み進められます。
-   **テキスト保持**: 読み込み画面と入力画面を行き来しても、入力したテキストは消えずに保持されます。
-   **レスポンシブデザイン**: 長い文章が表示される場合も、フォントサイズを極端に小さくすることなく、適切に折り返して表示します。
-   **多言語対応**: 日本語の分かち書きだけでなく、英語のスペース処理も最適化されています。

## 技術スタック

-   **フロントエンド**: React
-   **ビルドツール**: Vite
-   **スタイリング**: CSS Modules / Vanilla CSS

## セットアップ手順

### 前提条件

-   Node.js (v14以上)
-   npm (v6以上)

### インストール

1.  リポジトリをクローンします:
    ```bash
    git clone https://github.com/yourusername/shotread.git
    cd shotread
    ```

2.  依存パッケージをインストールします:
    ```bash
    npm install
    ```

3.  開発サーバーを起動します:
    ```bash
    npm run dev
    ```

4.  ブラウザで `http://localhost:5173` (またはターミナルに表示されたポート) を開きます。

## 使い方

1.  **テキスト入力**: 読みたい文章をテキストエリアにペーストします。
2.  **読書開始**: "Start Reading" ボタンをクリックします。
3.  **設定調整**:
    -   **Speed**: スライダーで表示時間（ミリ秒）を調整します。
    -   **Chunk**: スライダーで一度に表示する文字量（Lv.1〜Lv.5）を調整します。
4.  **操作方法**:
    -   `Space`: 再生 / 一時停止
    -   `→ (右矢印)`: 次へ進む (Manualモード)
    -   `← (左矢印)`: 前へ戻る (Manualモード)
